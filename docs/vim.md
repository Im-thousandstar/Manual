## vimrcの作成
```
$ vim ~/.vimrc
```
ここに必要に応じて書き込む

## Vim command
* 「V」(Shift + v)キーにより行選択のビジュアルモードに切り替え．
* 「G」(Shift + g)キーによりファイルの最後の行までまとめて選択．
* 「d」キーを入力すると、選択した範囲（ファイル内全て）を削除．
